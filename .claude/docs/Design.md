# Theme Specification

This document outlines the visual and interactive properties of our design system's theme.

## Color Palette

Our brand utilizes a vibrant color palette, designed for impact and clarity in a `light` environment.

*   **Primary Color:** `#059669` - Our core brand color, used for primary calls to action and key interactive elements.
*   **Secondary Color:** `#2563eb` - A complementary color, suitable for secondary actions, accents, and information hierarchy.
*   **Tertiary Color:** `#8b5cf6` - An additional accent color for highlights, badges, or decorative elements.
*   **Neutral Color:** `#0F172A` - A dark neutral color, serving as a base for backgrounds and text, ensuring high contrast in the light mode.

## Typography

The typography system is structured to provide a clear hierarchy and maintain readability.

*   **Headline Font:** `inter` - Used for all major headings, providing a distinct and modern feel.
*   **Body Font:** `inter` - Chosen for its legibility and versatility, ideal for paragraphs and longer text blocks.
*   **Label Font:** `inter` - Consistent with the body font for form labels and smaller UI elements, ensuring clarity.

## Shapes and Spacing

These properties define the physical attributes and layout of UI components.

*   **Roundedness:** `3` (Maximum, pill-shaped) - All UI elements feature maximum, pill-shaped roundedness, giving the interface a very soft and friendly appearance.
*   **Spacing:** `2` (Normal) - A balanced approach to whitespace, providing a comfortable visual separation between elements without feeling overly dense or sparse.
